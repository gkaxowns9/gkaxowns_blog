## 최솟값 만들기 문제 설명

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12941) 

- 리스트 두 개를 줬을 때, 각 값을 곱한 결과를 전부 더해서 나오는 최소값을 구하는 문제다
  자세한 설명을 위해 프로그래머스에서 설명을 복붙하자.

> 길이가 같은 배열 A, B 두개가 있습니다. 각 배열은 자연수로 이루어져 있습니다.
  배열 A, B에서 각각 한 개의 숫자를 뽑아 두 수를 곱합니다. 이러한 과정을 배열의 길이만큼 반복하며, 두 수를 곱한 값을 누적하여 더합니다. 이때 최종적으로 누적된 값이 최소가 되도록 만드는 것이 목표입니다. (단, 각 배열에서 k번째 숫자를 뽑았다면 다음에 k번째 숫자는 다시 뽑을 수 없습니다.)

- 그렇다고 한다

## 최솟값 만들기 접근 및 풀이

- 곱의 합의 최소값은 더 생각할 필요도 없이 큰 값은 작은 값이랑, 작은 값은 큰 값이랑 곱하면 된다.
  작성한 코드는 아래와 같다

```python
def solution(A,B):
    answer = 0

    A.sort(reverse=True)
    B.sort()
    for a, b in zip(A, B):
        answer += a * b

    return answer
```

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/70129)

- 어떤 문자열을 주면 정해진 연산을 계속 반복하여 얼마나 반복했는지를 출력하면 되는 문제다.

> 0과 1로 이루어진 어떤 문자열 x에 대한 이진 변환을 다음과 같이 정의합니다.

## 이진 변환 반복하기 접근 및 풀이

- 문제 설명에서 **이진 변환**에 대해 정의했고, 문자열 s가 "1"이 될 때까지 이를 반복한다고 했다.
  일반적인 반복도 좋지만, 제귀함수로 이 문제를 풀어보려고 한다. (개인적으로 단순한 계산이 아니면, 특정 조건까지 반복하는 문제는 제귀함수가 구현하기 쉬운 것 같다)

- 정의했던 이진 변환에서, 문자열의 길이 c를 이진법으로 표현한 문자열로 바꿔야 한다.
  파이썬의 `bin()` 함수로 객체를 바이너리 형태의 문자열로 바꿀 수 있다.
  대신 문자열 앞에 `0b`가 붙어서 슬라이싱으로 제거하여 해결했다.

- 또 0을 제거한 길이를 구해야 하는데, 이거는 기존 문자열의 길이와 0의 개수를 세서 서로 빼주는 형태로 구현했다.

- 제귀함수는 리턴값이 `[변환횟수, 제거된 0의 개수]` 형태로 되기에, 함수 안에서 호출한 함수의 리턴 값을 가져와 이번 제귀에서의 제거된 0의 개수와 변환횟수를 더하는 식으로 구현했다.

```python
def solution(s):
    if s == "1":
        return [0, 0]
    
    zore_count = 0
    
    for i in s:
        if i == "0":
            zore_count += 1

    a, b = solution(bin(len(s) - zore_count)[2:])

    a += 1
    b += zore_count
    return [a, b]
```
